# 常见查询模板类型

这是模板类型说明，不是现成 SQL 全集。用于帮助后续快速选择生成路径。

## 1. 名单筛选模板

适用：
- 外呼名单
- 升套名单
- 解约跟踪名单
- 携出用户筛选

特征：
- 主表筛用户
- 多个剔除条件
- left join 补状态、活动、渠道、费用
- 输出字段面向业务执行

## 2. 指标汇总模板

适用：
- 渗透率
- 融合率
- 流失率
- 收入 / 流量汇总

特征：
- 先构造分组维度
- 再统计用户数 / 收入 / 流量
- 常见拆分：王卡 / 非王卡、总体 / 分群

## 3. 月环比 / 次月追踪模板

适用：
- 上月 / 当月 / 次月状态追踪
- 解约用户后续去向
- 合约到期次月观察

特征：
- 多月 join
- 使用 `add_months` 与月末日期
- 输出阶段状态变化

## 4. 中间表构建模板

适用：
- 复杂任务
- 需要沉淀中间名单
- 要供下游导出或再次 join

特征：
- `create table ... as select`
- 多阶段过滤与打标
- 命名中常带日期/任务名

## 5. 推荐策略模板

适用：
- 升套 / 合约推荐
- 套餐档位归集
- 特邀低消 / 托底逻辑

特征：
- 强依赖业务规则
- 常需输出推荐档位、推荐原因、是否满足某类策略

## 使用建议

接到自然语言需求后，先判定最接近哪一类模板，再决定是：
- 写单条 SQL
- 写多步脚本
- 写中间表 + 结果表版本
