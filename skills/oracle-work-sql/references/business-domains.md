# 业务域总览

这套 Oracle SQL 语料主要来自真实工作任务，当前可稳定识别出以下业务域。

## 1. 流失 / 携出 / 解约追踪

典型任务：
- 携出用户取数
- 协议解约后状态追踪
- 解约后 30 天内携转 / 预约销户 / 销户分析
- 违约金、自动续约、新合约承接分析

常见输出：
- 用户状态
- 销户时间
- 合约开始/结束时间
- 次月状态
- 群组出账、收入、充值等跟踪字段

## 2. 升套 / 外呼 / 合约 / 维系任务

典型任务：
- 升套任务名单
- 外呼任务客群筛选
- 倍享 / 权益 / 低消 / 升 20 送 20 等策略
- 合约到期用户托底与例外处理

典型特征：
- 文件内业务注释很多
- 规则会按月份迭代
- 常伴随剔除清单、保留条件、推荐策略字段

## 3. B2I / 客群运营 / 渗透率分析

典型任务：
- B2I 用户整体指标分析
- 王卡 / 非王卡拆分
- 5G 渗透率、融合率、创新型产品渗透率
- 流量、收入、离网率等对比

## 4. 宽带 / 融合 / FTTR / 套餐营销

典型任务：
- 是否融合用户判断
- 是否有宽带 / 是否可营销宽带
- FTTR 办理、融合承接、宽带资源匹配
- 套餐与宽带速率映射

## 5. 名单型取数 / 中间表 / 临时表生成

典型任务：
- create table as select
- 多步中间表构建
- 用于后续外呼、投放、策略执行的名单产出
- 设备号、用户号、套餐、状态、渠道等字段明细输出

## 6. 收入 / 出账 / 流量 / 充值分析

典型任务：
- 计费收入、出账金额
- 免费流量 / 收费流量
- 累计充值、当月充值
- 欠费金额与欠费开始时间

## 使用建议

当用户提出需求时，先判断落在哪个业务域，再决定：
- 复用哪类高频表
- 复用哪类规则
- 输出明细名单、汇总指标还是追踪结果
